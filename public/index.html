Selamat Datang!

Jika kamu menjumpai situs ini, itu berarti situs ini mulai aktif.

Situs ini dikembangkan dengan menggunakan Termux.dev.
Web server yang digunakan adalah NGINX.
Situs ini berjalan pada backend yang dibuat dalam lingkungan javascript menggunakan framework ExpressJS.
Server backend dalam pengembangan situs ini menggunakan runtime JavaScript secepat kilat, yang disebut dengan BunJS.
Untuk dapat diakses secara online, situs ini mengandalkan Cloudflare.
Dan yang tak kalah penting, ini dihost dengan hanya menggunakan smartphone Android yang sudah usang!

Ini saja yang ingin saya tulis. Terima kasih telah mengunjungi situs ini!
